# Reader-s-Corner-BookStore-

Readers' Corner Bookstore Website

Welcome to the **Readers' Corner Bookstore Website**, a creative and fully responsive web design project developed by **Rhulani Mahlahlane**.  
This project showcases a user-friendly online bookstore with a modern pink aesthetic, elegant typography, and intuitive navigation — ideal for book enthusiasts.

---

## Overview

The **Readers' Corner** website allows users to:
- Explore featured and popular books  
- Browse through a visually appealing image gallery  
- Learn more about the bookstore and its story  
- Contact the store through a simple form  
- Send enquiries directly via a Google Form integration  

The design is fully responsive, ensuring an excellent experience on both desktop and mobile devices.

---

## Website Pages

| Page | Description |
|------|--------------|
| **Home (index.html)** | Welcoming introduction to Readers’ Corner with a hero section and featured books |
| **About (about.html)** | Background information about the bookstore |
| **Books (products.html)** | A display of book selections and descriptions |
| **Gallery (gallery.html)** | A grid of images featuring the bookstore and its collection |
| **Contact (contact.html)** | Contact details and location information |
| **Enquiry (enquiry.html)** | Embedded Google Form for customer enquiries |

---

## Design & Styling

- **Theme:** Soft pink tones, white backgrounds, and clean lines  
- **Typography:**  
  - *Headings:* Playfair Display  
  - *Body text:* Poppins  
- **Layout:** Consistent header and footer across all pages  
- **Design Goal:** To create a warm, welcoming online space that reflects the charm of a community bookstore
